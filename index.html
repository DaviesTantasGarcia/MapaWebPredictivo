Para disponer de un mapa web no es necesario comprar una licencia, solo es necesario conocer herramientas alternativas: "QGIS, TerrSet, HTML5 y CSS3", la interacción de tales herramientas resultará en un mapa web interactivo que muestra principalmente la predicción de pérdida de bosque para el año 2026 (para obtener la predicción se aplicó el siguiente modelo, clic sobre Enlace1).

Para conocer las métricas de las pérdidas de bosque en el tiempo (2016, 2021 y 2026) producto de la minería ilegal se creo un dashboard, para ello se usaron las herramientas de Excel y Looker Studio (clic sobre Enlace2 para ver el resultado). Ambos productos (mapa web interactivo y dashboard) permitirá a las jefaturas, entender, interpretar y decidir sobre los avances del mapeo de las variables (capas) trabajadas en beneficio de la entidad.
